Studtusk 🐘

Studtusk — це персоналізований менеджер завдань, розроблений спеціально для студентів. 
Це легка альтернатива складним інструментам на кшталт Notion, зосереджена на ефективному управлінні навчальним процесом.
🚀 Основні можливості
    Інтуїтивний таск-менеджер: Додавайте, редагуйте та видаляйте навчальні завдання за кілька кліків.
    Аналітика прогресу: Система моніторингу, що візуалізує кількість виконаних, поточних та протермінованих завдань.
    Центр нотаток: Швидке створення та зберігання важливої навчальної інформації в одному місці.
🛠 Технологічний стек
    Frontend: HTML5, CSS3 (використання Flexbox та Grid для адаптивного дизайну).
    Logic: JavaScript (ES6+).
    Backend/Database: MySQL для надійного зберігання даних користувачів.


English Version
Studtusk 🐘

Studtusk is a personalized task management tool designed specifically for students. It serves as a lightweight alternative to Notion, focused on efficient academic workflow management.
🚀 Key Features
    Intuitive Task Manager: Add, edit, and delete academic assignments with ease.
    Progress Tracking: A built-in system that displays the count of completed, in-progress, and pending tasks.
    Note-Taking System: A dedicated space to create and organize essential study notes.
🛠 Tech Stack
    Frontend: HTML5, CSS3 (leveraging Flexbox and Grid for responsive layouts).
    Logic: JavaScript (ES6+).
    Backend/Database: MySQL for secure and reliable data storage.
